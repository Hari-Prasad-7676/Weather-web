# Weather-web
A website to check weather of different cities 
Weather Application

A simple and responsive **Weather Application** that provides real-time weather information for any city using live API data.  
Built as part of a **hackathon / HSIP submission** to demonstrate frontend skills and API integration.

---

Features

Search weather by city name  
Displays temperature, humidity, and wind speed  
Shows current weather conditions with icons
Fast and lightweight  
Fully responsive (mobile + desktop)
